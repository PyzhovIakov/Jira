Инструкция по установке:

Скачайте приложение "iCRM" на свой телефон из магазина.

Ссылка на Google Play: https://goo.gl/Dg27oo

Ссылка на App Store: https://goo.gl/Sk1sgW



Если это первый запуск приложения:

  1. Откройте приложение.

  2. Выберите пункт: Настроить синхронизацию.

  3. В открывшемся окне выберите Вариант синхронизации «КОРП». Нажмите кнопку "Применить".

  4. В открывшемся окне выберите версию конфигурации, с которой настраивается синхронизация (Режим работы).

  5. В поле Код приглашения укажите код приглашения, для этого скопируйте его из письма.

  6. Если код приглашения указан верно, начнется загрузка данных. После завершения загрузки данных, с приложением можно начинать работать.

Если ранее вы уже запускали приложение и смотрели демо-пример:

  1. Перейдите на закладку "Синхронизация" - "Настройки синхронизации"

  2. В открывшемся окне выберите пункт «Подключить синхронизацию КОРП».

  3. В открывшемся окне выберите версию конфигурации, с которой настраивается синхронизация (Режим работы).

  4. В поле Код приглашения укажите код приглашения, для этого скопируйте его из письма.

  5. Если код приглашения указан верно, начнется загрузка данных. После завершения загрузки данных, с приложением можно начинать работать.



В случае возникновения вопросов по синхронизации, обратитесь к администратору корпоративного решения "1C:CRM"
