Управление поставляемыми панелями 1С:Аналитика

Обработка «Управление поставляемыми панелями 1С:Аналитика» (раздел Администрирование, группа Сервис) предназначена для упрощения установки поставляемых в составе конфигураций и расширений преднастроенных панелей 1С:Аналитика, включая диаграммы, дашборды, настройки рабочего стола, а также дополнительные источники данных.

Рабочее место поддерживает следующие функции: управление системой аналитики (указание адреса подключения к серверу для информационной базы), просмотр в иерархическом виде поставляемых панелей и их статус, выполнение действий по установке/обновлению, как с отдельными по поставляемыми панелями, так и со всеми вместе:

  • Адрес сервера 1С:Аналитика. Позволяет выполнить интеграцию сервера «1С:Аналитики» в опубликованное на веб-сервере приложение (аналог команды «Функции для технического специалиста – Управление системой аналитики»).
  • Поставляемые панели 1С:Аналитика. Иерархический список доступных в конфигурации (и расширениях конфигурации), поставляемых панелей 1С:Аналитика, с указанием поставщика – разработчика панели.
  • Статусы поставляемых панелей. Даты изменений поставляемых панелей в конфигурации (актуальные версии поставщика) и установленных на сервере «1С:Аналитка» панелей. На основании дат изменений определяются статусы панелей (список возможных статусов представлен в Легенде).
  • Действия с поставляемыми панелями. Возможное действие с отдельными поставляемыми панелями, которое можно выполнить с учетом текущего статуса (список возможных действий представлен в Легенде).
  • Групповые действия. Установка и обновление поставляемых панелей, представленных в списке. Установка выполняется для всех неустановленных поставляемых панелей, представленных в списке. Обновление – для всех установленных панелей, статус у которых указан как «неактуальная версия».

Для табличной части доступно выпадающее меню, возможность использования некоторых пунктов зависит от доступности связи с сервером «1С:Аналитика».

Режим разработчика

Обработка «Управление поставляемыми панелями 1С:Аналитика» предоставляет дополнительные возможности для разработчиков тиражных решений. Команды режима «Для раазработчика» собраны в меню формы «Еще» и позволяют:

  • Выгрузить поставляемую панель 1С:Аналитика. Позволяет выгрузить настройки текущего подключения в поставляемую панель 1С:Аналитика для последующего включения в макет конфигурации для тиражирования (более подробно - в руководстве пользователя).
