Обработка "Интеграция с WhatsApp"

Назначение

Служебная обработка, предназначена для обмена данными с мессенджером WhatsApp.

Интеграция с WhatsApp  позволяет обмениваться сообщениями с клиентами в  АРМ Диалоги.


Важно! Данная интеграции  доступна при наличие активного действующего  договора «1С:КП Отраслевой».


Важно! На текущий момент у мессенджера WhatsApp нет открытого официального API.  Поэтому интеграцию можно выполнить только через специальные платные сервисы, так называемые API‑шлюзы. В 1С:CRM для этого используется сервис ChatApi.
