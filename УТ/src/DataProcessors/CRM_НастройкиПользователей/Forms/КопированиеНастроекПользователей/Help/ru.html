Предназначена для копирования персональных настроек пользователей CRM.

Команда копирования персональных настроек CRM пользователей можно найти на панели Персональные настройки пользователя (CRM) раздела Сервис. 

Копирование настроек пользователей

  • В поле Скопировать настройки CRM пользователя выберите пользователя, чьи персональные настройки CRM будут использоваться для копирования другим пользователям:
    • Нажмите  Выбрать, укажите список Пользователи, в котором будет производиться копирование настроек.
    • В списке Выбор пользователя выделите нужного пользователя, нажмите кнопку Выбрать. Также можно дважды щелкнуть по нужному пользователю левой кнопкой мыши.
  • В поле Кому копировать выберите пользователей, которым необходимо скопировать настройки. Переключатель принимает значения:
    • Выбранным пользователям - становится доступен выбор пользователей. Перейдите по ссылке Выбрать.
      • В открывшемся окне "Выбор пользователей для копирования настроек" с помощью флажков выберите нужных пользователей.
      • После завершения выбора ссылка принимает значение: имя одного выбранного пользователя или количество выбранных пользователей.
    • Всем пользователям - после выбора значения можно перейти к следующему шагу.
  • В поле Разделы персональных настроек CRM выберите, какие настройки копировать другому пользователю или пользователям:
    • Каждый пункт этого поля соответствует разделам из панели Персональные настройки пользователя (CRM) раздела Сервис.
      • Если только выбран пункт "Настройки", значит будут скопированы только настройки из раздела "Настройки" панели Персональные настройки пользователя (CRM).
      • Если только выбран пункт "Оповещения", значит будут скопированы только настройки из раздела "Оповещения" панели Персональные настройки пользователя (CRM).
      • Если выбраны оба пункта, значит будут скопированы все настройки из этих двух разделов.
  • Для перехода к копированию настроек с выбранными параметрами нажмите Скопировать и закрыть, для того чтобы вернуться в программу или Скопировать, для того чтобы после завершения копирования этих настроек продолжить копирование.

Ограничения

  • Копирование персональных настроек CRM выполняет администратор.
