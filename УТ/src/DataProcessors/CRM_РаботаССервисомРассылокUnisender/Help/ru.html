Обработка "Интеграция с сервисом рассылок UniSender"

Назначение


Обработка предназначена для осуществления почтовых рассылок при помощи сервиса UniSender

Важно!
Система UniSender используется только для отправки рассылок, подготовленных с помощью документа Рассылка электронных писем. Его нельзя использовать для отправки обычных писем из почтового клиента.
