Обработка "Интеграция с Instagram"

Назначение

Служебная обработка, предназначена для интеграции с публичными страницами Instagram, позволяет обмениваться комментариями с клиентами в АРМ Диалоги и регистрировать новые Интересы клиентов.

Важно! Данная интеграции доступна при наличие активного действующего договора «1С:КП Отраслевой».

Важно! Общение ведется только через комментарии в публикациях, общение через direct не предусмотрено из-за ограничений официального API Instagram.
